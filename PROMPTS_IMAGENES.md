## NOTAS

- Pide siempre formato horizontal
- Si una imagen sale con texto incrustado, regenera pidiendo "no text, no words, no letters, no logos"
- Después de descargar, comprime con [TinyPNG](https://tinypng.com) para reducir peso
- Nombra cada archivo exactamente como se indica arriba
- Coloca todo en la carpeta `public/` del proyecto
